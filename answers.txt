Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. It is platform-independent due to its write once, run anywhere capability.

Object-Oriented,Platform Independent,Robust and Secure,Multithreaded,High Performance (via JIT compiler),Portable and Distributed

A constructor is a special method used to initialize objects. It has the same name as the class and does not have a return type.

The JVM is a runtime environment in Java that executes bytecode. It acts as an interpreter between the compiled Java code and the machine code of the operating system.

ArrayList: Uses a dynamic array, faster for accessing elements. while LinkedList: Uses a doubly linked list, faster for insertions and deletions.
